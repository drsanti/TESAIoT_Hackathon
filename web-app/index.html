Bitstream Studio Web App Examples

Self-contained HTML pages for the Bitstream telemetry provider (ws://127.0.0.1:9997) and MQTT broker (ws://127.0.0.1:8883) via /@bitstream/ SDK assets.

How to run

  1. Install Bitstream Studio from the vsix/ folder in this repo (VS Code → Extensions → Install from VSIX).
  2. Start backend services (bridge + telemetry provider) — automatic on extension activate, or run Bitstream Studio dev bridge if you build from source.
  3. Toolbar Server → Start broker for MQTT examples (ex09–ex15).
  4. Command Palette → Serve Web App Folder over HTTP → select this web-app folder (serves pages + /@bitstream/ws-live-data.js and /@bitstream/mqtt-live-data.js).
  5. Open the catalog URL from the notification (or browse to index.html).
  6. Connect a device (toolbar Bitstream + COM) or external Simulator (toolbar Simulator) so samples flow to the provider.
