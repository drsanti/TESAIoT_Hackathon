# Lab 03 — Open the GATT link

## Goal

Connect GATT, discover BS_RX / BS_TX / BS_LINK, and read the link snapshot.

## On-screen steps

1. **Connect** via the shared session.
2. Confirm three characteristics.
3. Read / re-read **BS_LINK**.

## Acceptance

Linked session + all three chars + link snapshot → chapter complete.

## Notes

On Windows, pair `TESAIoT-*` in Settings → Bluetooth if connect drops in a few ms. Use `/diag` for isolated bring-up. See `docs/WEB_BLUETOOTH_WINDOWS.md`.
