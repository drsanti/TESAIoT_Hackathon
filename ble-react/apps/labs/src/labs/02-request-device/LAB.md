# Lab 02 — Pick the board

## Goal

Practice the Web Bluetooth device chooser with a user gesture.

## On-screen steps

1. Click **Request device**.
2. Select a peripheral advertising `TESAIoT-*`.

## Acceptance

Device name/id shown → chapter complete.

## Notes

This chapter does **not** open GATT (Lab 03 does). Always use a fresh picker — BLE addresses can rotate (RPA). If the board is missing, RESET until soft-blue advertising.
