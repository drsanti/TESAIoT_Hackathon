Bitstream Studio Web App Examples

Self-contained HTML pages that consume live sensor data from the Bitstream Telemetry Provider (ws://127.0.0.1:9997) via the @bitstream/live-data SDK.

How to run

  1. Install Bitstream Studio from the vsix/ folder in this repo (VS Code → Extensions → Install from VSIX).
  2. Start backend services (bridge + telemetry provider) — automatic on extension activate, or run Bitstream Studio dev bridge if you build from source.
  3. Command Palette → Serve Web App Folder over HTTP → select this web-app folder (serves pages + /sdk/live-data.js).
  4. Open the catalog URL from the notification (or browse to index.html).
  5. Connect a device (toolbar Bitstream + COM) or external Simulator (toolbar Simulator) so samples flow to the provider.

Offline fallback: copy live-data.browser.js to vendor/live-data.js in this folder and use any static server (npx serve . from web-app/).

Per-sensor

  • ex01
    SHT40 — Temperature & Humidity
    Reference template: catalog labels, bars, dual-series chart, stale handling.
  • ex02
    DPS368 — Pressure & Temperature
    Barometric pressure bar + temperature + pressure history chart.
  • ex03
    BMM350 — Magnetometer & Compass
    Mag X/Y/Z values + 2D compass needle from atan2(magY, magX).
  • ex04
    BMI270 — IMU (Accel + Gyro)
    Six-axis bars and sparklines for acceleration and gyroscope.
  • ex05
    BMI270 — Orientation / Artificial Horizon
    Euler or quaternion → pitch/roll horizon canvas visualization.
